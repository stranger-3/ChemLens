About ChemLens

ChemLens is a chemical intelligence web application designed to provide quick and accurate information about chemical compounds. You can search by name, formula, or SMILES string.

This project demonstrates frontend & backend logic using only HTML, CSS, and JavaScript. It includes over 50 compounds in the database, each with overview, properties, uses, and structure.

Designed with a modern glassmorphism style using TailwindCSS for a clean and responsive interface.
